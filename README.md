# DSQTools
拿来练手的超超超简单辅助工具（造轮子来咯）

纯正前端实现:P，最纯粹的网页三件套（顶多多个 NodeJS），给予你最原始的享受。

## 当前实现功能
- Trm Icon 元素可视化
- 兑换条件可视化并图形化编辑  
  目前支持条件：`diplay.shiny`、`actions` 的各个条件和列表
- 兑换条件同步（待实现）

## 快速开始
### 前置条件
- 已安装 Node.js

### 安装与启动
1. 修改 `index.html` 下的包导入路径
2. 安装依赖
   ```bash
   npm install
   ```
3. 启动服务（默认端口 3000）
   ```bash
   node app.js
   ```

### 更省事的方式
如果只想快速浏览页面：  
在 VSCode 安装 `live-server`，直接在 `public` 目录启动即可，`index` 都不用改。
